# CineGenome : Hybrid Movie Recommender System

Welcome to the **CineGenome** — a powerful blend of **Content-Based Filtering (CBF)** and **Collaborative Filtering (CF)** enhanced with **Natural Language Processing (NLP)** techniques to deliver personalized and relevant movie suggestions.

---

## 🚀 Overview

This project combines the strengths of multiple recommendation strategies to overcome their individual limitations:

* 🎯 **Content-Based Filtering (CBF)** — recommends movies based on similarity in content (genres, descriptions, metadata).
* 🤝 **Collaborative Filtering (CF)** — leverages user behavior and preferences to suggest movies.
* 🧠 **NLP Integration** — processes textual data such as movie plots, keywords, and tags to improve recommendation quality.

By hybridizing these approaches, the system provides more accurate, diverse, and context-aware recommendations.

---

## 🧩 Key Features

✨ Hybrid recommendation engine (CBF + CF)
✨ NLP-powered feature extraction from movie metadata
✨ Scalable and modular design
✨ Evaluation using ranking-based metrics
✨ Easily extendable for real-world applications

## 📊 Evaluation Metrics

To measure recommendation quality, the system uses:

### 🔹 Precision@K

Evaluates how many of the top-K recommended movies are relevant.

### 🔹 MRR@K (Mean Reciprocal Rank)

Measures how early the first relevant item appears in the recommendation list.
